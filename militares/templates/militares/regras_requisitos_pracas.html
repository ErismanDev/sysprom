Regras de Requisitos para Promoção por Antiguidade - Praças

Informações Importantes
  • Todas as promoções de praças são por antiguidade
  • Os requisitos de cursos são cumulativos (cada promoção exige os cursos da anterior + novos)
  • Para Soldado → Cabo: aceita CFSD OU Curso de Formação de Praças
  • Interstício mínimo e inspeção de saúde em dia são obrigatórios
Transição de Posto Cursos Obrigatórios Observações

 Transição para oficial
Requisitos Gerais
  • Situação ativa no serviço
  • Inspeção de saúde em dia
  • Interstício mínimo completado
  • Numeração de antiguidade informada
  • Cursos inerentes ao posto subsequente
Observações
  • Os cursos são cumulativos
  • Não é necessário ter ficha de conceito para promoção por antiguidade
  • A ordenação é por numeração de antiguidade (menor número = mais antigo)
  • Em caso de empate na antiguidade, considera data de promoção
Como Funciona a Ordenação

Para promoção por antiguidade de praças:

  1. 1º Critério: Quadro (Praças sempre por último)
  2. 2º Critério: Posto (hierarquia: ST → 1S → 2S → 3S → CAB → SD)
  3. 3º Critério: Numeração de antiguidade (menor número = primeiro)
  4. 4º Critério: Data de promoção (mais antiga = primeiro)
