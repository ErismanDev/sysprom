==========================================
COMANDOS PARA ATUALIZAR CHAT NO DIGITAL OCEAN
==========================================

OPÇÃO 1: USAR SCP PARA FAZER UPLOAD
------------------------------------
No seu computador Windows (PowerShell ou CMD):

# Navegar até a pasta do projeto
cd C:\projetos\Sysgabom

# Fazer upload dos arquivos (substitua por sua chave SSH)
scp static/js/chat-widget-ios.js seprom@64.23.185.235:/home/seprom/sepromcbmepi/static/js/
scp static/js/chat-tempo-real.js seprom@64.23.185.235:/home/seprom/sepromcbmepi/static/js/
scp static/js/chat-calls.js seprom@64.23.185.235:/home/seprom/sepromcbmepi/static/js/

# OU usando WinSCP (interface gráfica):
# 1. Conecte-se ao servidor
# 2. Navegue até: /home/seprom/sepromcbmepi/static/js/
# 3. Arraste os 3 arquivos para o servidor


OPÇÃO 2: COPIAR CONTEÚDO MANUALMENTE
-------------------------------------
No servidor Digital Ocean (SSH):

# Conectar ao servidor
ssh seprom@64.23.185.235

# Navegar até o diretório
cd /home/seprom/sepromcbmepi

# Fazer backup dos arquivos atuais
cp static/js/chat-widget-ios.js static/js/chat-widget-ios.js.backup
cp static/js/chat-tempo-real.js static/js/chat-tempo-real.js.backup
cp static/js/chat-calls.js static/js/chat-calls.js.backup

# Editar os arquivos (copie o conteúdo dos arquivos atualizados)
nano static/js/chat-widget-ios.js
# Cole o conteúdo, salve: Ctrl+O, Enter, Ctrl+X

nano static/js/chat-tempo-real.js
# Cole o conteúdo, salve: Ctrl+O, Enter, Ctrl+X

nano static/js/chat-calls.js
# Cole o conteúdo, salve: Ctrl+O, Enter, Ctrl+X


APÓS ATUALIZAR OS ARQUIVOS
---------------------------
# Ativar ambiente virtual
source venv/bin/activate

# Coletar arquivos estáticos (se necessário)
python manage.py collectstatic --noinput

# Reiniciar Gunicorn
sudo systemctl restart gunicorn

# Verificar se reiniciou corretamente
sudo systemctl status gunicorn

# Ver logs se houver erro
sudo journalctl -u gunicorn -n 50 --no-pager


VERIFICAR SE FUNCIONOU
-----------------------
1. Acesse o site: http://64.23.185.235
2. Abra o chat
3. Verifique se as imagens aparecem nas mensagens
4. Teste uma chamada de vídeo/voz
5. Limpe o cache do navegador (Ctrl+Shift+R)


RESTAURAR BACKUP (se necessário)
---------------------------------
# Se algo der errado, restaure os backups:
cp static/js/chat-widget-ios.js.backup static/js/chat-widget-ios.js
cp static/js/chat-tempo-real.js.backup static/js/chat-tempo-real.js
cp static/js/chat-calls.js.backup static/js/chat-calls.js

# Reiniciar
sudo systemctl restart gunicorn
